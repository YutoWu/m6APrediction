# m6APrediction

An R package for predicting m6A modification sites from genomic features using a random forest machine learning model.

## Installation

You can install the development version of m6APrediction from GitHub with:

```r
# Install devtools if not already installed
if (!require("devtools")) {
  install.packages("devtools")
}

# Install m6APrediction
devtools::install_github("YutoWu/m6APrediction")
